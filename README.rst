C Project
=========

Name: ygclass-rename

Description
===========

Rename a gclass

For example:

You have a gclass *Prot_http* :

    Prot_http (c_prot_http.c, c_prot_http.h files)

And you want reuse the code in a new gclass *Prot_http2* ::

    yclass-rename prot_http prot_http2
